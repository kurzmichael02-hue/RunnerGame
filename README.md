Entwickelt als DHBW-Projekt 

## Tools & Engine

- **Godot** (geplant)
- **Git & GitHub** (Versionsverwaltung)
- **Branch-Workflow** mit `main` / `dev` / `feature-*`
- **Sound & Musik:** *FL Studio 2025*, *Audacity*, *Serum (Plugin)*
